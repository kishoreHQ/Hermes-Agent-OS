# Security

Target controls (enforced as platform matures):

- Unified credential broker (handles only to runtimes)  
- Trust labels on memory  
- Sandbox tiers on runtimes  
- Agent modes: Full / Assist / Observe  
- Signed plugin manifests (later phase)  
- Deny-by-default external actions under Assist  

Security reviews are mandatory in the execution workflow before merge of boundary-touching changes.
