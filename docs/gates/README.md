# Gates

Product acceptance gates for Hermes-Agent-OS.

| Gate | Intent |
|------|--------|
| H0 | Foundation docs + kernel skeleton |
| H1 | Host API parity smoke |
| H2 | Plugin load + example provider/runtime |
| H3 | Mission Control against Hermes kernel |
| H4 | Interchangeability proof (≥2 providers, ≥2 runtimes) |
| H5 | Security + production readiness |

AESP-RI GATE-1…9 remain authoritative for the reference monorepo until Hermes owns those demos.
